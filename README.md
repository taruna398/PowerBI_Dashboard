# PowerBI_Dashboard

📱 Mobile Sales Dashboard | Power BI

🚀 Project Overview

An interactive Power BI dashboard designed to analyze mobile sales performance across different dimensions like time, location, and product categories.
This project helps in tracking business growth, trends, and performance comparisons.

📊 Dashboard Views

🟢 1. Overall Sales Dashboard
✔ Displays total revenue, quantity, and transactions
✔ Highlights top-performing brands & models
✔ Shows city-wise and customer insights

📅 2. MTD (Month-To-Date) Dashboard
✔ Tracks current month performance
✔ Monitors daily sales trends
✔ Helps in short-term decision making

📈 3. Same Period Last Year Dashboard
✔ Compares current vs previous year performance
✔ Identifies growth trends 📊
✔ Analyzes seasonal patterns

🎯 Key Objectives

✔ Analyze total sales and transactions
✔ Identify top-performing products
✔ Track monthly and yearly trends
✔ Compare current vs past performance

🛠️ Tools & Technologies

💻 Power BI
📊 Excel (Data Cleaning & Preparation)

📌 Key Insights

✨ Top brands contribute majority of revenue
✨ UPI/online payments dominate transactions
✨ Sales show seasonal growth patterns
✨ Certain cities drive higher sales performance
